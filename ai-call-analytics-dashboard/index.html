What this dashboard shows

          

            The dashboard provides a concise view of call outcomes for AI voice agents that handle appointment bookings.
            It focuses on the core metrics that matter for performance:
          

          
    
            
  • Call volume – how many leads the agent is speaking to.
    • 
            
  • Confirmed bookings – how many calls successfully convert.
    • 
            
  • Reschedules – leads who stay warm but move to a new time.
    • 
            
  • No answers – missed opportunities that may need follow-up automation.
    • 
          

        


        

          
Role in an AI calling workflow

          

            The dashboard sits on top of call logs from AI voice agents and turns them into an at-a-glance operational view.
          

          
    
            
  • AI agent places and receives calls with leads.
    • 
            
  • The calling platform records outcomes and basic statistics.
    • 
            
  • A JSON export or API endpoint provides aggregated metrics.
    • 
            
  • This dashboard renders those metrics as cards, charts and raw data for review.
    • 
          

          

            In a production environment the JSON would typically be returned by an HTTP API rather than a local file,
            but the visual layer remains the same.
