# Dev Projects

A collection of small, focused projects built by Devansh Dubey between 2023 and 2025.

- **Zypher AI – Automation Suite (2025)** – Multi-agent demo for appointment confirmation, lead qualification and call analytics with a FastAPI mock backend.
- **AI Call Analytics Dashboard (2025)** – Front-end dashboard showing call performance from JSON data.
- **Smart Inventory Tracker (2024)** – Python tool to forecast when stock will run out.
- **Email Summariser & Action Extractor (2024)** – Flask app that condenses long emails and pulls out action points.
- **Password Strength Auditor (2023)** – Simple security tool that scores and explains password strength.
